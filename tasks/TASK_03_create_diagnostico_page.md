# TASK 03 — Crear página /diagnostico

## Objetivo
Crear una página `/diagnostico` para capturar leads calificados.

## Debe incluir
- Explicación breve del diagnóstico.
- Qué revisa Kikuba.
- Para quién es.
- Formulario con campos de calificación.
- CTA claro.
- Mensaje de privacidad simple.
- Qué pasa después.
- Mensaje de confirmación.

## Campos
- Nombre
- Nombre del negocio
- Ciudad
- Giro del negocio
- Cómo recibes clientes hoy
- Dónde sientes más desorden
- Qué usas para organizarte
- Qué te gustaría mejorar primero
- Preferencia de contacto
- Contacto
- Link a sitio/redes, opcional

## Backend
Si no hay backend, dejar el formulario preparado para conectar a Tally, Google Forms, Formspree o Netlify Forms.

## Criterio de terminado
La página permite que una persona solicite diagnóstico y que Kikuba pueda registrar ese lead en CRM.
