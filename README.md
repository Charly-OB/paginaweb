# Juan Carlos Obeso | Web & Datos

Landing page estatica para presentar servicios de desarrollo web, SaaS a la medida y analisis de datos comercial.

## Publicacion con GitHub Pages

Repositorio: `Charly-OB/paginaweb`

1. En GitHub, entra a `Settings > Pages`.
2. Selecciona `Deploy from a branch`.
3. Elige la rama `main` y carpeta `/root`.
4. Guarda los cambios y espera a que GitHub genere la URL publica.
